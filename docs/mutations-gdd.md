# Мутации — GDD v1

Скелет забега: **класс-новичок** (старт) + **спутник** (старт) → **3 опоры** (оружие / грудь / ботинки) + **рюкзак** (основная сила) → **форма R8** → **мутация R16**.

## Правила подсчёта

- **Очки ветки** за prep перед боем:
  - тег на **манекене**: ×2
  - тег в **рюкзаке** (top-8 предметов по «силе»): ×1
  - **спутник** (bias): ×1.5 к совпадающим тегам
  - **слот усиления** (голова / грудь / ботинки): ×2 к семейству слота
- **R8:** лидер ≥40% от max по классу → первая форма + слабое правило.
- **R16:** лидер ≥55% → мутация + капстоун. При ничьей — приоритет спутника → опора груди → ботинки.
- **Супер-новичок класса:** мутация **ОТШЕЛЬНИК / УНИВЕРСАЛ** — нет доминанты (лидер <35%), спутник **Странник**, 4+ семейств тегов.

Капстоуны дают **правило**, не плоский +50% урона. Бюджет капстоуна ≈ 5–8% эффективности билда.
