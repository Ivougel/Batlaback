# Набор предметов под новую сборку (чертёж)

Статус: **PR-A** — усиления 1×1 в рюкзаке; магазин ~22% с R2.

## Три слоя силы (финальная формула)

| Слой | Доля | Где |
|------|------|-----|
| Рюкзак + синергии | 80–88% | сетка 9×7 |
| Опоры: оружие + 3 слота усилений | 8–12% | манекен + `head/chest/boots` |
| Класс-новичок + спутник + мутация R16 | 5–10% | старт + milestone |

**Усиление** занимает **1×1 в рюкзаке**. Нет свободной клетки — эффект не активен. Полоска «Усиления» — только отображение.

## Слоты усилений

| Слот | Открытие | Роль | Источники |
|------|----------|------|-----------|
| `head` | R2 | ось / стиль | магазин, ключ |
| `chest` | R6 | выживание | крафт, магазин |
| `boots` | R11 | темп, капстоун | крафт, редкий магазин |

Код: `systems/enhancements.js`, UI: полоска «Усиления» в prep HUD.

## Три канала получения (когда предметы готовы)

1. **Магазин** — отдельная карточка «Усиление» или тег `enhancement` в ролле.
2. **Крафт** — рецепт виден при тегах в рюкзаке + нужная мутация ≥ форма R8.
3. **Ключ** — 1×1 в рюкзаке открывает ветку (`unlock_build` в meta), не занимает слот усиления.

## Типы предметов в новом наборе (~70 шт. целевой пул)

### A. Усиления (`ENHANCEMENT_CATALOG`) — 24 шт.

- 8 головы, 8 груди, 8 ботинок.
- Поля: `slot`, `families[]`, `recommendedMutations[]`, `acquisition[]`, `combat` (слабые правила).

### B. Усилители рюкзака — ~18 шт.

Предметы 1×1–1×2 с эффектом:

- `amplifySlot: chest | boots | head`
- `amplifyFamily: fire | holy | poison | …`
- `amplifyEquip: staff | wand | twoHand`

Подсветка клеток в prep (этап после предметов).

### C. Ключи веток — ~12 шт.

`metaEffect: { type: "unlock_build", build: "triple_pyro_mage" }` — bias магазина, видимость крафтов.

### D. Опоры-оружие / броня манекена — из основного каталога

Часть существующих предметов помечается `recommendedTriple`; новые godly — под тройки.
