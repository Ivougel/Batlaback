# Отчёт: описания для игроков 16+

**Парадигма:** понятный русский, без dev-жаргона (`×0.1`, «прок», «стак», «сокет»), короткие предложения, бой → магазин.

**Обновлено:** 143 предмета (55 уже были ок).

## Принципы

- «Вампиризм» → «лечит на X% от урона»
- «Прок» → «чаще срабатывают эффекты»
- «Стак» → «бонус» / конкретное имя (удача, блок, жар)
- «4с» → «4 сек»
- «(раз)» → «один раз за бой»
- Камни: «В оружии / аксессуаре / броне: …»
- Магазин: «В магазине …», без «съедает»

## Примеры

| Предмет | Описание |
|---------|----------|
| Камень зарядов | При активации: +1 мана. |
| Деревянный щиток | При блоке или уклонении (30% шанс): блокирует 7 урона… Блок 5. |
| Нестабильный перекомбобулятор | +12 удачи. Каждые 4 сек: случайно… В магазине перерабатывает… |
| Тим | При попадании: 50% крадёт 1 бонус у врага… |

Повтор: `node tools/humanize-item-descriptions.mjs --apply`
